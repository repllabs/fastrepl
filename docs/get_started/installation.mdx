## Installation

### Basic
```bash
pip install "fastrepl"
```

### With RAGAS
```bash
pip install "fastrepl[ragas]"
```

### With Langfuse
```bash
pip install "fastrepl[langfuse]"
```

You can find all releases [here](https://pypi.org/project/fastrepl).


## Setup
```python
import fastrepl
```
You don't need to memorize module structure. Single import is all you need.
